Cascading Style Sheets (CSS) is properly done with a .css file listing all CSS commands for your website.
Another way to do CSS is using the style tag at the top head part of a webpage.
Here is a typical in-document style: 
    b { 
      color: #00f00;
    }

I do CSS inline inside the tag. It overpowers all other methods.
